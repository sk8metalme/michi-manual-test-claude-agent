# Product Overview

ブラウザベースのクラシックテトリスゲームを提供するWebアプリケーション。プレイヤーはPC（デスクトップ）ブラウザでテトリミノを操作し、ライン消去を通じてスコアを獲得する。保守性と拡張性の高いオニオンアーキテクチャの実装例としても機能する。

## Core Capabilities

- **ゲームプレイ**: テトリミノの移動・回転・落下、ライン消去、スコア計算
- **難易度システム**: レベル進行に応じた落下速度の自動調整（レベル1-10）
- **永続化**: スコア履歴の保存と過去10件の表示（高スコア順）
- **リアルタイム操作**: WebSocketによる低レイテンシ通信（100ms以内のレスポンス目標）

## Target Use Cases

### プレイヤー向け
- カジュアルなブラウザゲームプレイ（匿名、アカウント不要）
- 過去のスコア記録の確認と自己ベスト更新

### 開発者向け
- オニオンアーキテクチャ（ヘキサゴナルアーキテクチャ）の実装パターン学習
- ドメイン駆動設計（DDD）とクリーンアーキテクチャの実践例

## Value Proposition

- **アーキテクチャ品質**: ドメインロジックを純粋Javaで実装し、外部フレームワークから完全分離。テスト容易性と保守性を最大化
- **シンプルで楽しい**: クラシックテトリスのルールに忠実な、直感的なゲーム体験
- **教育的価値**: 実用的なアプリケーションを通じて、ヘキサゴナルアーキテクチャの原則を体験
